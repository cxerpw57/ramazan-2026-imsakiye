- **Ramazan:** 1 – 29 Ramazan 1447
- **Şehir:** İstanbul, Türkiye

- 29 günlük namaz vakitleri tablosu (İmsak, Güneş, Öğle, İkindi, Akşam, Yatsı)

- Kadir Gecesi (14 Mart) özel olarak işaretlenmiştir
